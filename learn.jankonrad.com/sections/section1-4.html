Firmware a jeho úloha

Firmware je speciální software uložený v čipu na základní desce počítače. Tento software je uložen ve flash paměti ROM a slouží ke spuštění a inicializaci hardwaru počítače. Poté předá řízení operačnímu systému.

Paměť ROM vs RAM
  • ROM (Read-Only Memory): Nepřepisovatelná paměť, kde je uložen firmware. Po vypnutí napájení data zůstávají.
  • RAM (Random Access Memory): Dočasná operační paměť, ve které běží aplikace a operační systém. Po vypnutí se její obsah vymaže.
  • 🧠 Hierarchie témat: BIOS a UEFI jako klíčové komponenty firmware.
    • BIOS: Starší typ firmware, který umožňuje základní nastavení a inicializaci zařízení.
    • UEFI: Novější nástupce BIOSu s moderní grafikou, podporou velkých disků a vyšší bezpečností.

    BIOS

    BIOS je zkratka pro „Basic Input Output System“. Jedná se o první program, který se spustí po zapnutí počítače.

    Hlavní funkce BIOSu
    • Inicializace hardwaru
    • POST test (Power-On Self Test)
    • Nastavení bootovací sekvence
    • Nastavení hodin, zabezpečení a základního systému
    💡 Shrnutí

    BIOS je jednoduché prostředí bez myši. Je omezený velikostí, obvykle 1 MB nebo méně.

    UEFI

    UEFI znamená „Unified Extensible Firmware Interface“. Jedná se o moderní verzi firmware, která nahradila BIOS.

    Výhody UEFI
    • Podpora větších disků (více než 2 TB)
    • Grafické rozhraní a ovládání myší
    • Lepší bezpečnost díky funkci Secure Boot
    • Možnost síťového bootování (PXE)
    💡 Shrnutí

    UEFI je moderní a výkonnější nástroj pro řízení startu počítače než BIOS.
